I asked ChatGPT  "how can i create simple symbols for cards in my card game" It responded with several steps on how to create symbols for cards, and it responded with a great workflow on making symbols which gave me an idea on how to make symbols for my cards alongside this it.
It also gave me a few recommendations for software to make my art in these include:
	- Inkscape 
		- which is a dedicated software for vector graphics like symbols or icons as well as being open source and universal (meaning it can run on Windows, Linux and MacOS)
	- Aseprite
		- This is more pixel art focused however at high resolutions it can be a great tool for making pixel art
	- Krita
		- Similarly to Inskape Krita is free and open source with universal support however it more generalised for art and can be overkill for making icons like ChatGPT says
From my judgement these are all top tier suggestions for making symbols and icons for my game however I have never used any apps before except for Inkscape which I used for Linux ricing however it is not enough experience for making icons for my game so as an alternative I asked ChatGPT "what about photoshop" it responded that it is an options however there can be several points of resistance and can be a bit awkward for making icons with, but because I do not have the time to learn a new software like Inkscape or Adobe Illustrator to make new icons I decided to use Photoshop in the end from my judgement I have the most experience with Photoshop meaning it would be significantly easier to make my games 2D art with, however if I had more time I would instead try learning Inkscape or Illustrator as they both interest me plus I have a small experience with Inkscape and already know a decent amount of the Adobe suite so I believe it would not take a long time to learn the software.
