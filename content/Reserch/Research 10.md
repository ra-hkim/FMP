# UE5
## Nav mesh overlay
I searched on google "how to see nav mesh in ue5" as I wanted to see the where the enemies can path find too without using the AI overlay that I already knew about which requires the game to be running.
![[Pasted image 20260514121800.png]]
Google's AI responded to press the "P" key to toggle an overlay which shows where the enemies/AI can navigate, the Google AI also states that the overlay can be activated inside the editor without starting the game as well as inside the game simulation, this will deferentially help debug AI issues and especially help when adding foliage and obstacles to my game.
