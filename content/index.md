# Quartz 4
I originally wrote my blog using obsidian notes instead as I found writing with markdown formatting significant easier than using using google sites/docs formatting obsidian notes faster workflow meant that I could spend more time programming and less time messing around with google sites. I then re-uploaded everything to google sites to comply with the course, however __you are more then welcome to read through these files__.

I decided to upload my obsidian notes using quartz 4 which is a tool designed to convert markdown files into html files, which also has dedicated support for obsidian note's extra features. Go to [[Making this blog]] to learn how I uploaded this to the internet.
